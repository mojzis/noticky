A few czech songs, mostly "Koledy" - Christmas Carols, made so that it would be easy for kids (or anyone) to play, with propper "fingerings" - the numbers above the notes indicating which finger to use for each note.

Where necessary & meaningful, the melody is transposed to c key to make it easier to play.

Work in progress! :)
please note that both the engravings and this page should eventually improve :).
In case you want to add some, check out the github repo.
