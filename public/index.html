A few czech songs, mostly Christmas Carrols, made so that it would be easy for kids (or anyone) to play, with propper "fingerings" - the numbers above the notes indicating which finger to use for each note.

Where necessary, the melody is transposed to c key to make it easier to play.

  • Byla cesta byla ušlapaná
  • Narodil se Kristus Pán
  • Nesem vám noviny
  • Pásli ovce valaši
  • Skákal pes
  • Veselé vánoční hody

Links lead to pdf files, a page / file for each song, made so that it would be as easy as possible to read while playing.
